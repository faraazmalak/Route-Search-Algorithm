To get all possible routes between 2 towns, please do the following:

1. Click on File->Read Towns to load the file with locations of the towns  (Please select locations.txt from FileChooser)

2. Create roads between towns with mouse. Stop dragging the mouse when the white circle turns yellow. (White circle will turn yellow when the Red circle and white circle intersect while dragging)

3. Enter indices of start and end locations and click on Get Routes button

4. Shortest Route will be highlighted in blue color
